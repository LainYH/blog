# 第2章 物理层

## 2.1 通信基础

### 2.1.1 基本概念

| 概念 | 说明 |
|------|------|
| **数据** | 传送信息的实体 |
| **信号** | 数据的电气/电磁表现 |
| **码元** | 用一个固定时长的信号波形表示一位 k 进制数字 |
| **信源** | 产生和发送数据的源头 |
| **信宿** | 接收数据的终点 |
| **信道** | 信号的传输媒介 |

#### 信号的分类

| 类型 | 说明 |
|------|------|
| **模拟信号/连续信号** | 取值连续 |
| **数字信号/离散信号** | 取值离散（0 和 1） |

#### 通信方式

| 方式 | 说明 |
|------|------|
| **单工通信** | 只能单向传输（如广播） |
| **半双工通信** | 双向但不能同时（如对讲机） |
| **全双工通信** | 双向同时传输（如电话） |

### 2.1.2 信道的极限容量 ⭐

#### 奈奎斯特定理（无噪声信道）

$$C = 2W\log_2 V \quad (b/s)$$

其中：
- W 为信道带宽（Hz）
- V 为信号的状态数（码元种类数）

⚠️ **在无噪声情况下，数据率受限于码元速率（2W）。**

#### 香农定理（有噪声信道）⭐

$$C = W\log_2 (1 + S/N) \quad (b/s)$$

其中：
- W 为信道带宽（Hz）
- S/N 为信噪比（信号功率/噪声功率）

⚠️ **信噪比常用分贝（dB）表示：** dB = 10log₁₀(S/N)

⚠️ **香农定理给出的是理论上限。实际数据率受奈奎斯特和香农双重约束。**

**例：** 信道带宽 3kHz，信噪比 30dB，求最大数据率。

- S/N = 10^(30/10) = 1000
- C = 3000 × log₂(1+1000) ≈ 3000 × 9.97 ≈ **30kb/s**

### 2.1.3 编码与调制 ⭐

#### 数字数据 → 数字信号（编码）

| 编码方式 | 说明 | 特点 |
|---------|------|------|
| **不归零编码（NRZ）** | 高电平=1，低电平=0 | 无法同步（连续相同数据时） |
| **不归零反转编码（NRZI）** | 遇到1跳变，遇到0不跳变 | 解决了连续1的同步问题 |
| **曼彻斯特编码** ⭐ | 每个码元中间跳变，高到低=0，低到高=1 | 自同步，带宽加倍 |
| **差分曼彻斯特编码** | 每个码元中间跳变，边界有跳变=0，无跳变=1 | 自同步，抗干扰好 |

⚠️ **曼彻斯特编码是 408 高频考点！**

#### 数字数据 → 模拟信号（调制）

| 调制方式 | 说明 |
|---------|------|
| **幅移键控（ASK）** | 改变载波的振幅 |
| **频移键控（FSK）** | 改变载波的频率 |
| **相移键控（PSK）** | 改变载波的相位 |
| **正交幅度调制（QAM）** | ASK 和 PSK 的结合 |

#### 模拟数据 → 数字信号（数字化）

**脉冲编码调制（PCM）：** 采样 → 量化 → 编码

⚠️ **采样定理：** 采样频率 ≥ 2 × 信号最高频率

#### 模拟数据 → 模拟信号（调制）

频分复用（FDM）等。

### 2.1.4 数据交换方式 ⭐

#### （1）电路交换

建立物理连接 → 数据传输 → 释放连接

| 优点 | 缺点 |
|------|------|
| 时延小、实时性好 | 建立连接时间长、线路利用率低 |
| 有序传输、没有冲突 | 不同速率和格式的终端无法通信 |

⚠️ **电路交换三阶段：建立连接 → 数据传输 → 释放连接**

#### （2）报文交换

以报文为单位，采用**存储转发**方式。

| 优点 | 缺点 |
|------|------|
| 无需建立连接、动态分配线路 | 报文大小不限，需较大存储空间 |
| 提高线路利用率 | 转发时延较长 |

#### （3）分组交换 ⭐⭐

将报文分成若干分组，采用**存储转发**方式。

| 分类 | 说明 |
|------|------|
| **数据报方式** | 每个分组独立路由，可能乱序到达 |
| **虚电路方式** | 先建立虚电路，分组沿同一路径传输，有序到达 |

⚠️ **分组交换 vs 电路交换：**
- 分组交换：存储转发，线路利用率高，存在存储转发时延
- 电路交换：独占线路，实时性好，线路利用率低

⚠️ **数据报 vs 虚电路：**

| 对比 | 数据报 | 虚电路 |
|------|--------|--------|
| 连接建立 | 不需要 | 需要 |
| 路由选择 | 每个分组独立选择 | 建立时选择，所有分组走同一路径 |
| 分组顺序 | 可能乱序 | 有序到达 |
| 可靠性 | 由端系统保证 | 由网络保证 |
| 适应性 | 强 | 差（链路故障则整条虚电路断） |

## 2.2 传输介质

### 2.2.1 导向传输介质

| 介质 | 特点 |
|------|------|
| **双绞线** | 最常用，有屏蔽和非屏蔽之分 |
| **同轴电缆** | 抗干扰性好，常用于有线电视 |
| **光纤** ⭐ | 传输损耗小、抗干扰、带宽大；单模光纤远距离，多模光纤近距离 |

### 2.2.2 非导向传输介质

| 介质 | 特点 |
|------|------|
| **无线电波** | 全方向传播，如 Wi-Fi、蜂窝网络 |
| **微波** | 方向性好，直线传播，需要中继 |
| **红外线** | 方向性强，不能穿透障碍物 |

## 2.3 物理层设备

| 设备 | 功能 |
|------|------|
| **中继器（Repeater）** | 放大和再生数字信号，扩展传输距离 |
| **集线器（Hub）** | 多端口的中继器，工作在物理层 |

⚠️ **中继器/集线器不能连接不同速率的网段，不能隔离冲突域。**

⚠️ **5-4-3 规则：** 以太网中最多 5 个网段、4 个中继器、3 个网段挂接计算机。

## 2.4 本章例题

### 例题1：奈奎斯特和香农

某信道带宽 4kHz，信噪比 30dB。
（1）按香农定理求最大数据率
（2）若信号有 4 种状态，按奈奎斯特求最大数据率

**解析：**
（1）S/N = 10^(30/10) = 1000，C = 4000 × log₂(1001) ≈ 4000 × 10 = **40kb/s**
（2）C = 2 × 4000 × log₂4 = 2 × 4000 × 2 = **16kb/s**
取两者较小值为实际最大数据率约束。

### 例题2：曼彻斯特编码

画出比特流 10110010 的曼彻斯特编码波形。

**规则：** 每个码元中间跳变，高到低 = 0，低到高 = 1

| 比特 | 1 | 0 | 1 | 1 | 0 | 0 | 1 | 0 |
|------|---|---|---|---|---|---|---|---|
| 编码 | ↑ | ↓ | ↑ | ↑ | ↓ | ↓ | ↑ | ↓ |

（↑ 表示低到高跳变表示1，↓ 表示高到低跳变表示0）

### 例题3：分组交换时延

设报文长度为 M bit，分组长度为 (L+h) bit（L 为数据，h 为分组头），共分为 n 个分组，经过 k 段链路，每段链路传输速率为 R b/s。求总时延。

**解析：**
- 发送时延：n × (L+h)/R + (k-1) × (L+h)/R（流水线效应）
- 传播时延：k × d（d 为每段传播时延）
- 总时延 = (n+k-1)(L+h)/R + kd
